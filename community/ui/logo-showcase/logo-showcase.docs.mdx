A showcase of logos with gentle animation. Lazy loading is added by default for all images but can be overridden with the `loading` prop.

### Client logos from imported Json

Imports mock data in json format from the entity component which contains an array of logos.

```js
import { clientLogos } from '@teambit/community.entity.images';

<LogoShowcase logos={clientLogos} />;
```

### Showcase of Logos with array of objects

```js live
<LogoShowcase
  images={[
    {
      src: 'https://static.bit.dev/homepage-enterprise-logos/Dell.svg',
      alt: 'dell'
    },
    {
      src: 'https://static.bit.dev/homepage-enterprise-logos/Carlsberg-1.svg',
      alt: 'Carlsberg'
    }
  ]}
/>
```
